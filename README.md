## About

Simple gallery on HTML/CSS/JS/NodeJS

You need installed NodeJS to work with project:
* [Node.js](https://nodejs.org)

### Usage

Install NPM packages:
  ```sh
  npm install
  ```

Run host server:
  ```sh
  start node server.js
  ```
  
Follow this link:
  ```
  localhost:8000
  ```
Easy.
